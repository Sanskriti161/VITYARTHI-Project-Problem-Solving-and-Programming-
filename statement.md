# Problem Statement & Scope

## Problem Statement
Managing personal finances manually can be frustarting and may not always be correct. Users especially students in college need a secure, reliable banking system to create accounts, perform transactions, and maintain records. A digital banking application provides a safe platform for account management with security through its PIN verification and transaction history like features.

## Scope of the Project

The project includes:
- Banking Account creation with secure PIN authentication
- Account login and verification
- Deposit and withdrawal of money
- Real-time balance checking
- Complete transaction history provided
- Account information management
- Security features like PIN verification

## Target Users
- Individuals especially college students wanting to manage their finances.
- Students learning banking concepts.

## High-Level Features

| Feature | Description |
|---------|-------------|
| **Account Creation** | Create new accounts with PIN and initial deposit |
| **Secure Login** | Authenticate users with account number and PIN |
| **Deposit** | Add money to accounts with transaction logging |
| **Withdrawal** | Withdraw money with balance verification |
| **Balance Check** | View current account balance anytime |
| **PIN Verification** | Verify account security PIN |
| **Account Info** | View complete account details |
| **Transaction History** | View all account transactions |
| **Error Handling** | Comprehensive validation and error messages |
